IMPORTANT PRE-INSTALLATION INSTRUCTIONS

To ensure a smooth installation process, please verify the following:

Active Internet Connection: A stable internet connection is required for initial device authentication.

Machine Readiness: Ensure your Flex or Laser machine is powered ON and the communication cable (Serial/USB) is securely connected to this PC.

Administrative Rights: Please run this setup as an Administrator to allow the agent to register as a background service.

Close Other Applications: To avoid port conflicts, please close any other printer monitoring tools before proceeding.
